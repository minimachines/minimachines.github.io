My Alexa Cafe Experience

Monday

On Monday we were inroduced to the basic html and css format
and layout. We also learned some simple html commands
like br ans hr. We were given time to explore the html and css commands
and their outcomes on a page.

Tuesday

Tuesday was the first work day for our websites. I decided
to make a website about dance and I strated implementing
my ideas into code. I also learned to link my css commands
to decorate my page.

Wednesday

On Wednesday I learned how to make a navigation bar and drop
down menu for my website. I started creating different informational
pages about different dance realted topics. I also learned how to
embed videos from YouTube onto my website and group things together
using div.

Thursday

Thursday was the day for finishing touches on the websites. I completed
all the needed pages for the navigation bar and I verified all my code. I also
learned how to make a Magic 8 Ball using jacascript arrays and the random Math
function. I also added a horoscope page and a navigation bar to the Magic 8 Ball
website.

Favorite Camp Memory

My favorite experience from this week was being able share about something
that I feel passionate for. I really enjoyed the feeling of being able to control my
own project and create whatever I wanted without any limits. It felt really rewarding
to look back at what I created after putting is so much hard work and time.
